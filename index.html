TSP Algorithms Visualizations

We have displayed visualizations of several inputs of data. We are using data from the TSPLIB as input and the Hamiltonian paths we create as outputs. We are focusing on the visualizations of the paths created by the Nearest Neighbor, Greedy, Convex Hull, and Christofides algorithms. The last visualization corresponds to that of the optimal path. Code for the computation of the paths and visualizations can be found here.
For the graphs, the % difference is calculated by algorithm path length/ optimal path length.
